AI-Powered Email Generator for Recruiters

  • This project demonstrates a practical application of Generative AI in automating professional communication.
  • I developed an AI-based web application that generates recruiter-targeted emails by analyzing a candidate’s resume and the job description (or company career page).
  • The app intelligently extracts relevant information from both inputs and formulates a concise, personalized email suitable for job applications.
  • The system is built using LangChain for orchestration, with Groq-hosted LLM APIs (e.g., LLaMA, Mistral) for generation.
  • The workflow includes resume parsing, job context understanding, and prompt templating. The interface was built using Streamlit for rapid prototyping.
  • This project highlights my experience in LLM integration, text preprocessing, chain construction, and leveraging AI to improve job-seeking automation.

Solar Power Prediction

  • This project focuses on building a machine learning pipeline to accurately predict solar panel power generation using meteorological data.
  • I performed extensive Exploratory Data Analysis (EDA) on key environmental features including temperature, humidity, solar irradiance, and time-of-day.
  • After preprocessing and cleaning the dataset, I engineered relevant features to capture patterns in power production.
  • Multiple regression models were trained and evaluated including Linear Regression, Decision Trees, and Gradient Boosting (XGBoost) with hyperparameter tuning for performance optimization.
  • The final model achieved strong predictive accuracy and was evaluated using RMSE and R² metrics.
  • The project demonstrates proficiency in regression modeling, feature engineering, and real-world energy data analysis with practical applications in sustainable energy forecasting and optimization.

Intelligent Product Recommendation System

  • This project focuses on developing a hybrid recommendation engine capable of delivering personalized product suggestions in an e-commerce environment.
  • It includes both content-based filtering and clustering-based segmentation for improved accuracy and relevance.
  • I performed text vectorization (TF-IDF, CountVectorizer) and text cleaning on product descriptions and user reviews
  • Customer segments were identified using unsupervised learning (K-Means, Hierarchical Clustering) to provide more targeted recommendations.
  • Cosine similarity and Euclidean distances were used for ranking top-N products. The pipeline was optimized for scalability and modularity.
  • This solution simulates a real-world recommender service for online retail platforms, and demonstrates my skills in text analytics, clustering algorithms, and recommender system architecture.
